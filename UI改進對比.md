## 操作流程對比

### 舊版流程
```
1. 點擊棋盤 → 立即落子 ❌ 容易誤觸
2. 無法預覽
3. 無法取消
```

### 新版流程
```
1. 點擊棋盤 → 顯示預覽 ✅
2. 確認位置正確
3. 點擊「著手」按鈕 → 確認落子 ✅
4. 可以重新選擇位置（點擊其他格子）
```

## 視覺改進

### 棋子效果
```
舊版：平面圓形
新版：立體漸層 + 陰影

●  →  ◉  (黑棋：徑向漸層)
○  →  ◎  (白棋：徑向漸層)
```

### 預覽效果
```
選擇位置後：
┌─┬─┬─┐
│ │◉│ │  ← 半透明 + 脈衝動畫
├─┼─┼─┤     + 綠色指示點
│ │ │ │
└─┴─┴─┘
```

### 最後一手標記
```
┌─┬─┬─┐
│ │●│ │  ← 紅點標記
├─┼─┼─┤     在棋子上方
│ │🔴│ │
└─┴─┴─┘
```

## 響應式斷點

```
📱 手機 (< 480px)
├─ 棋格：22px
├─ 字體：0.65rem
└─ 全螢幕佈局

📱 大手機/小平板 (481-768px)
├─ 棋格：26px
├─ 字體：0.75rem
└─ 全螢幕佈局

💻 桌面 (> 768px)
├─ 棋格：32px
├─ 字體：0.85rem
└─ 居中佈局 + padding
```
